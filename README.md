# 🎨 Pizarra Digital PRO+ v2.3

Pizarra digital colaborativa para clases virtuales. Hasta 50 participantes. Compatible con Zoom/Meet. Offline.

## ✨ Características
- Colaboración en tiempo real (WebRTC) — hasta 50 participantes
- Anfitrión puede expulsar participantes
- Toolbar horizontal/vertical
- Lápiz, resaltador, borrador, texto, notas adhesivas editables
- Líneas, flechas, 12 figuras 2D, 6 figuras 3D
- Importar PDF para anotar encima
- Líneas guía ajustables (puntos)
- Exportar PNG, PDF, JSON
- Grabar clase (pantalla + audio)
- Navegador web integrado + nueva pestaña
- Calculadora, modo presentación, pantalla completa
- PWA instalable, funciona offline, auto-guardado
