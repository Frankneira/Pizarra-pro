# 🎨 Pizarra Digital PRO+ v2.1 | Colaborativa

Pizarra digital colaborativa (hasta 50 participantes). Compatible con Zoom/Meet. Funciona offline.

## Características
- Colaboración en tiempo real (WebRTC/PeerJS)
- El anfitrión puede expulsar participantes
- Herramientas: lápiz, resaltador, formas, texto, importar PDF
- Exportar PNG/PDF, grabar clase, modo presentación
- PWA instalable, funciona sin internet

## Deploy
**Vercel:** Importa este repo → Deploy.
**GitHub Pages:** Settings → Pages → main → Save.
