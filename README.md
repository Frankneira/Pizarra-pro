# 🎨 Pizarra Digital PRO+ | Edición Docente 2026

Pizarra digital profesional para clases virtuales. 100% gratuita, funciona offline y es compatible con Zoom, Google Meet y proyección.

## ✨ Características

- **Herramientas de dibujo**: Lápiz, resaltador, borrador con colores y grosor ajustable
- **Formas**: Líneas, flechas, figuras 2D (12 tipos), figuras 3D (6 tipos)
- **Texto y notas**: Texto libre, notas adhesivas
- **Importar PDF**: Abre PDFs directamente sobre la pizarra para anotar encima
- **Páginas múltiples**: Crea y navega entre páginas ilimitadas
- **Líneas guía**: Patrón de puntos ajustable
- **Exportar**: PNG, PDF multipágina, proyecto JSON
- **Grabar clase**: Graba pantalla con audio (micrófono + sistema)
- **Puntero láser**: Para presentaciones en vivo
- **Calculadora**: Integrada para cálculos rápidos
- **Modo presentación**: Optimizado para Zoom/Meet (pantalla completa, sin distracciones)
- **Offline**: Funciona sin conexión a internet (PWA)
- **Auto-guardado**: Almacena tu trabajo automáticamente en el navegador

## 🚀 Despliegue en GitHub Pages

1. Haz fork de este repositorio
2. Ve a **Settings** → **Pages**
3. En **Source** selecciona `main` y carpeta `/ (root)`
4. Clic en **Save**
5. Tu pizarra estará disponible en `https://tu-usuario.github.io/pizarra-digital-pro/`

## 💻 Uso Local

Solo abre `index.html` en tu navegador. No requiere instalación ni servidor.

## ⌨️ Atajos de Teclado

| Atajo | Acción |
|-------|--------|
| `P` | Lápiz |
| `H` | Resaltador |
| `E` | Borrador |
| `V` | Seleccionar |
| `T` | Texto |
| `Espacio` | Mover pizarra |
| `Ctrl+Z` | Deshacer |
| `Ctrl+Y` | Rehacer |
| `Ctrl+C/V/X` | Copiar/Pegar/Cortar |
| `Delete` | Eliminar selección |
| `Esc` | Cerrar paneles |
| `F11` | Pantalla completa |

## 🛠️ Tecnologías

- [Fabric.js](http://fabricjs.com/) — Motor de canvas
- [jsPDF](https://github.com/parallax/jsPDF) — Exportación PDF
- [Tailwind CSS](https://tailwindcss.com/) — Estilos
- [Font Awesome](https://fontawesome.com/) — Iconos
- Service Worker — Soporte offline (PWA)
